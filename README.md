# Simple Translator 🌍

A basic web-based translator application built to convert text from one language to another using a translation API. This project focuses on creating a simple, fast, and responsive user experience.

## Features

* Translate text between multiple languages
* Simple and clean user interface
* Quick translation output
* Responsive design for desktop and mobile
* Easy language selection

## Tech Stack

* HTML
* CSS
* JavaScript
* Translation API

## Purpose

This project was built to practice API integration, DOM manipulation, and frontend development while creating a practical real-world application.

## Installation

Clone the repository:

```bash id="p0w0j7"
git clone <repository-link>
```

Open the project folder and run the `index.html` file in your browser.

## Usage

* Enter text in the input box
* Choose source language
* Choose target language
* Click translate
* View the translated result instantly
